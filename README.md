🚀 Maaş Günü Pro (MGP)
Maaş Günü Pro, finansal özgürlük yolculuğunuzu otomatiğe bağlayan, 7 adımlı metodoloji üzerine kurulu, modern ve şık bir kişisel finans yönetim platformudur. Apple tarzı minimalist arayüzü ve dinamik grafik analizi ile bütçenizi yönetmek hiç bu kadar keyifli olmamıştı.

✨ Öne Çıkan Özellikler
7 Adımlı Strateji: Finansal zemini oluşturmaktan, acil durum fonuna, borç kartopundan yatırım yönetimine kadar uzmanlaşmış bir akış.

Dinamik Grafik Analizi: Kaydettiğiniz verileri aylık ve yıllık bazda karşılaştıran interaktif sütun ve donut grafikler.

Veri Kalıcılığı: Tüm verileriniz yerel bir SQLite veritabanında güvenle saklanır; istediğiniz zaman geçmiş dönemlere göz atabilirsiniz.

Modern Tasarım: Açık renkli (Light Mode), profesyonel ve kullanıcı dostu "Apple Style" arayüz.

Geniş Zaman Aralığı: 2026'dan 2050'ye kadar tüm finansal planlamalarınızı kapsar.

🛠️ Kullanılan Teknolojiler
Frontend: Vue.js 3, Tailwind CSS, ApexCharts

Backend: Node.js, Express.js

Veritabanı: SQLite3

🚀 Kurulum (Local Setup)
Projeyi bilgisayarınızda çalıştırmak için şu adımları izleyin:

Depoyu Klonlayın:

Bash
git clone https://github.com/kullaniciadin/maas-gunu-pro.git
cd maas-gunu-pro
Bağımlılıkları Yükleyin:

Bash
npm install
Sunucuyu Başlatın:

Bash
node server.js
Erişim:
Tarayıcınızda http://localhost:3000 adresine gidin.

🚀 Salary Day Pro (English)
Salary Day Pro is a modern and sleek personal finance management platform built on a 7-step methodology designed to automate your journey to financial freedom.

✨ Key Features
7-Step Strategy: A specialized flow from building a financial foundation to emergency funds and investment management.

Dynamic Analytics: Interactive bar and donut charts that compare monthly and yearly financial data.

Data Persistence: Your data is securely stored in a local SQLite database for future reference.

Professional UI: A clean, "Apple-style" light-mode interface focused on user experience.

🛠️ Tech Stack
Vue.js 3, Tailwind CSS, ApexCharts, Node.js, Express, SQLite3
